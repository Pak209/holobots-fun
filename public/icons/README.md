# Navigation Icons

Please upload your SVG files for the navigation here.

## Expected SVG Files:
- `dashboard.svg` - Dashboard/Home icon
- `arena.svg` - Arena/Battle icon
- `sync.svg` - Sync Training icon
- `marketplace.svg` - Marketplace icon

## Notes:
- SVGs should follow the mecha-fitness aesthetic (angular, tech-inspired)
- Icons will be styled with cyan (#33C3F0) as default and yellow (#FFC627) when active
- Recommended size: 24x24 or scalable vector format
